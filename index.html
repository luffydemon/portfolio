Compétences BUT Informatique

Les 6 compétences du référentiel BUT

Compétence 1

Réaliser un développement d'application

Développer des applications informatiques simples en utilisant des langages de programmation appropriés.

Niveau : Débutant
  • Programmation en Python et Java
  • Concepts de base de la POO
  • Développement d'interfaces graphiques simples

Compétence 2

Optimiser des applications informatiques

Appréhender et construire des algorithmes efficaces pour optimiser les performances des applications.

Niveau : Débutant
  • Analyse de la complexité algorithmique
  • Structures de données de base
  • Optimisation de code simple

Compétence 3

Administrer des systèmes informatiques communicants

Installer, configurer et dépanner des systèmes informatiques dans des contextes d'entreprise.

Niveau : Débutant
  • Bases de l'administration Linux
  • Configuration réseau simple
  • Installation de services web

Compétence 4

Gérer des données de l'information

Concevoir et exploiter des bases de données et optimiser les données d'un système d'information.

Niveau : Débutant
  • Modélisation de bases de données
  • Requêtes SQL simples et complexes
  • Normalisation des données

Compétence 5

Conduire un projet

Identifier les besoins métiers des clients et y apporter des réponses informatiques adaptées.

Niveau : Débutant
  • Gestion de projet Agile (Scrum)
  • Analyse des besoins utilisateurs
  • Planification et suivi de projet

Compétence 6

Travailler dans une équipe informatique

Acquérir, développer et mobiliser ses aptitudes pour travailler efficacement dans une équipe informatique.

Niveau : Débutant
  • Travail collaboratif
  • Communication technique
  • Participation aux projets de groupe
